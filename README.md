# 📊 Customer Churn Analysis

## 📌 Overview

Customer churn is a major challenge for businesses, as losing customers directly impacts revenue.
This project performs **Exploratory Data Analysis (EDA)** to understand customer behavior and identify the key factors that contribute to churn.

---

## 🎯 Objectives

* Analyze customer data to identify churn patterns
* Understand factors influencing customer retention
* Provide actionable business insights

---

## 📂 Dataset

* Contains customer demographics, account details, and service usage
* Target variable: **Churn (Yes/No)**

---

## 🛠️ Tools & Libraries

* pandas – data manipulation
* NumPy – numerical operations
* Matplotlib – basic visualization
* Seaborn – advanced visualization

---

## 🔍 Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Visualization
5. Insights & Recommendations

---

## 📊 Key Insights

* Customers with **month-to-month contracts** have higher churn
* **New customers** (low tenure) are more likely to leave
* Higher **monthly charges** increase churn risk
* Certain **payment methods** show higher churn behavior

---

## 💡 Business Recommendations

* Encourage long-term contracts with incentives
* Improve onboarding experience for new customers
* Offer flexible pricing for high-risk customers
* Target churn-prone segments with retention campaigns
